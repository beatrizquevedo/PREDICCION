Introduccin

El conjunto de datos a utilizar en este trabajo se ha construido utilizando la puntuacin media en Ciencias por pas del Programa para la Evaluacin Internacional de Estudiantes (PISA) 2006, junto con el GNI per cpita (paridad del poder adquisitivo, dlares de 2005), el ndice educativo, el ndice de salud y el ndice de desarrollo humano de la ONU (HDI).

Las variables clave son las siguientes:

  • Overall Science Score (average score for 15 year olds)
  • Interest in science
  • Support for scientific inquiry
  • Income Index
  • Health Index
  • Education Index
  • Human Development Index (composed of the Income index, Health Index, and Education Index)

Objetivo

El obejtivo de este trabajo es modelizar la relacin entre la puntuacin media (OSS) y el resto de variables, utilizando modelos de splines y GAM aplicando la validacin cruzada cuando sea necesario.

GAM

Se crea un primer modelo GAM para predecir la puntuacin media usando splines suavizados con los grados de libertad obtenidos y graficamos el comportamiento de las variables independientes.

gamsplines <- gam(overall ~ s(interest, 4.75) + s(support, 2) + s(income, 4.24) + s(health, 2) + s(edu, 2) + s(hdi, 8.6), data = pisa)

As, se observa que se comportan de manera ms o menos lineal health y edu.

par(mfrow = c(3, 2))
plot(gamsplines, se = TRUE, col = 'orange', lwd = 1)

Se utilizar otro modelo GAM para predecir la puntuacin media usando splines suavizados de las variables independientes, pero esta vez sin indicar sus grados de libertad.

gam <- gam(overall ~ s(interest) + s(support) + s(income) + s(health) + s(edu) + s(hdi), data = pisa)

Graficamos el modelo por variables independientes, y se puede apreciar cmo la linealidad de health y edu ha empeorado.

par(mfrow = c(3, 2))
plot(gam, se = TRUE, col = 'purple', lwd = 1)

Se crea un tercer modelo GAM pero con las variables que parecen ser ms lineales.

gamln <- gam(overall ~  s(health, 2) + s(edu, 2), data = pisa)
par(mfrow = c(2, 1))
plot(gamln, se = TRUE, col = 'green', lwd = 1)

Se realiza un anlisis de la varianza (ANOVA) para decidir entre gam, gamsplines y gamln.
