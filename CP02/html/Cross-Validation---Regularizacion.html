M√©todos de contracci√≥n

Se ajustar√° un modelo que contiene todos los p predictores usando t√©cnicas que restringen o regularizan las estimaciones de coeficientes, o equivalentemente, que reduce las estimaciones de coeficientes hacia cero. Estas t√©cnicas son la regresi√≥n de cresta (Ridge), el lazo (Lasso) y la red el√°stica (Elastic Net).

Regresi√≥n de cresta

Como ya se ha dividido la muestra anteriormente, se crean modelos matriciales y vectores respuesta para el training y el testing, descartando a sy vez el itercepto.

data_train_y <- log(data_train$salary)

data_test_x <- model.matrix(salary ~ ., data_test)[, -1]
data_test_y <- log(data_test$salary)

Red El√°stica

Por √∫ltimo, esta t√©cnica de penalizaci√≥n incorporta la selecci√≥n de variables de Lasso y la contracci√≥n de predictores correlacionados de Ridge.

lasso    <- glmnet(data_train_x, data_train_y, alpha = 1.0) 
elastic1 <- glmnet(data_train_x, data_train_y, alpha = 0.25) 
elastic2 <- glmnet(data_train_x, data_train_y, alpha = 0.75) 
ridge    <- glmnet(data_train_x, data_train_y, alpha = 0.0)

par(mfrow = c(2, 2), mar = c(6, 4, 6, 2) + 0.1)

plot(elastic2, xvar = "lambda", main = "Elastic Net (Alpha = .75)\n\n\n")

Una vez creado el tibble (con 10 alfas, de 0.0 a 0.9 y en intervalos de 0.1) se rellena la tabla y se obtienen los valores de alfa/lambda correspondientes.

for(i in seq_along(tuning_grid$alpha)) {
  fit <- cv.glmnet(data_train_x, data_train_y, alpha = tuning_grid$alpha[i], foldid = fold_id)
  tuning_grid$mse_min[i]    <- fit$cvm[fit$lambda == fit$lambda.min]
  tuning_grid$mse_1se[i]    <- fit$cvm[fit$lambda == fit$lambda.1se]
  tuning_grid$lambda_min[i] <- fit$lambda.min
  tuning_grid$lambda_1se[i] <- fit$lambda.1se
}

Se comprueba en el gr√°fico que a medida de aumenta el valor de alfa, se reducen los valores del MSE m√≠nimo. A√∫n as√≠, el error es m√≠nimo para alfa de 0 a 1, por lo que cualquier modelo ser√≠a v√°lido. Por tanto, el modelo que se utilizar√° es el Lasso porque a diferencia del Ridge no coge todas las variables y es m√°s sencillo.
